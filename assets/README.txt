PHOTO DU HERO DE LA PAGE D'ACCUEIL
==================================

Déposez ici la photo de fond du hero, sous le nom EXACT :

    assets/hero.jpg

C'est le chemin attendu par index.html (règle CSS .hero-photo).

Conseils :
- Format paysage (large), au moins 1600 px de largeur.
- JPEG optimisé, idéalement moins de 400 Ko (pour un chargement rapide).
- Le point d'intérêt (les visages) doit être vers le haut/centre :
  la photo est cadrée sur "center 28%".

Comportement :
- Photo présente  -> elle apparaît en fond du hero, fortement assombrie,
  avec le réseau de particules jaunes animé par-dessus.
- Photo absente   -> le hero reste propre (fond noir + particules),
  aucune image cassée n'est affichée.

Une fois le fichier déposé, prévenez-moi : je le committe et le pousse en ligne.
